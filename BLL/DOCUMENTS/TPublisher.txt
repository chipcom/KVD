TPublisher - класс строки описания организации выдавшей документ удостоверяющий личность
			связан с файлом s_kemvyd.dbf

Свойства:
	Name - Наименование организации выдавшей документ
Методы:
	New( nId, cName, lNew, lDeleted ) - создать объект
		где:
		nId - идентификатор объекта ( номер записи в файле БД )
		cName - наименование организации выдавшей документ
		lNew - признак существования объекта в файле БД
		lDeleted - признак пометки на удаление объекта в файле БД
	Save() - записать объект в БД
	Clone() - создать копию объекта
	getByID ( nID ) - получить объект по идентификатору
		где:
		nId - идентификатор объекта ( номер записи в файле БД )
	GetList() - получить список объектов из БД
	MenuPublisher() - получить список наименований организаций выдавших документы

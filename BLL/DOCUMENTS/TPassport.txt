TPassport - класс описывающий документ удостоверяющий личность

Данные:
	aMenuType - список описывающий типы документов
Свойства:
	DocumentType - тип документа удостоверяющего личность
	DocumentSeries - серия документа
	DocumentNumber - номер документа
	IDIssue - идентификатор организации выдавшей документ
	DateIssue - дата выдачи
	AsString - представление документа по установленной форматной строке
	Format - форматная строка вывода представления документа (TYPE - тип документа, SSS - серия, NNN - номер, ISSUE - кто выдал, DATE - дата выдачи)
Методы:
	New( nType, cSeries, cNumber, nIDIssue, dIssue ) - создать объект
		где:
		nType - тип документа
		cSeries - серия документа
		cNumber - номер документа
		nIDIssue - идентификатор организации
		dIssue - дата выдачи
